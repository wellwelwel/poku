# Virtual DOM

Let's simule a **Virtual DOM** (Browser) by using [**jsdom**](https://github.com/jsdom/jsdom), then testing its results with **Poku**:

```ts
import { assert } from 'poku';
import { JSDOM } from 'jsdom';

const dom = new JSDOM('<div id="name">Poku</div>');
const document = dom.window.document;

const name = document.querySelector('#name');

assert.strictEqual(name?.textContent, 'Poku', 'My div contains a Poku üê∑');
```
