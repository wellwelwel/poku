# `debug`


> `poku(targetDirs: string | string[], configs?: Configs)`
>
> `debug: boolean`

> _Since **1.4.0**_

By default **Poku** doesn't shows logs that doesn't comes from **Poku**'s **`assert`**, but you can enable them:

## API (_in-code_)

```ts
poku(['...'], {
  debug: true
});
```

## CLI

```bash
npx poku --debug ./test
```
