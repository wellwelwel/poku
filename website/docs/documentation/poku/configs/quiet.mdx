# `quiet`

> `poku(targetDirs: string | string[], configs?: Configs)`
>
> `quiet: boolean`

Perform tests with no logs. <br />
This option overwrites all `log` settings by exiting with code and no logs (see bellow).

## API (_in-code_)

```ts
poku(['...'], {
  quiet: true,
});
```

## CLI

> _Since **1.3.1**_

```bash
npx poku --quiet ./test
```
