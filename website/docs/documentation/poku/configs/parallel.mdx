# `parallel`

> `poku(targetDirs: string | string[], configs?: Configs)`
>
> `parallel: boolean`

## API (_in-code_)

```ts
/**
 * @default
 *
 * Sequential mode
 */

poku(['...'], {
  parallel: false,
});
```

```ts
/**
 * Parallel mode
 */

poku(['...'], {
  parallel: true,
});
```

## CLI

```bash
# Parallel mode

npx poku --parallel ./test
```
