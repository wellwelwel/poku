# `failFast`

By using `failFast`, **Poku** will stop the tests at the first failure.

<History
  records={[
    {
      version: '2.5.0',
      changes: [
        <>
          <strong>CLI:</strong> deprecate <code>--fail-fast</code> in order to{' '}
          <code>--failFast</code>.
        </>,
      ],
    },
  ]}
/>

## CLI

```bash
npx poku --failFast ./test
```
