# `failFast`

> `poku(targetPaths: string | string[], configs?: Configs)`
>
> `failFast: boolean`

By setting `failFast` to `true`, **Poku** will stop the tests at the first failure.

## API (_in-code_)

```ts
poku(['...'], {
  failFast: true,
});
```


## CLI

```bash
npx poku --fail-fast ./test
```
