# `timeout`

Sets the maximum time in milliseconds that each test file is allowed to run.

## CLI

```bash
npx poku --timeout=5000 ./test
```

## Config File

<Tabs groupId="poku-config">
  <TabItem default value='js' label='poku.config.js'>

```ts
import { defineConfig } from 'poku';

export default defineConfig({
  timeout: 5000,
});
```

  </TabItem>
  <TabItem value='json' label='.pokurc.jsonc'>

```json
{
  "$schema": "https://poku.io/schemas/configs.json",
  "timeout": 5000
}
```

  </TabItem>
</Tabs>

<hr />

:::tip
Default timeout is `undefined` (Infinite).
:::
