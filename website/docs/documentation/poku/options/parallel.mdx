# `parallel`

Run the tests files in parallel.

<History
  records={[
    {
      version: '2.2.0',
      changes: [
        <>
          Support for <code>-p</code> short flag.
        </>,
      ],
    },
  ]}
/>

## CLI

```bash
# Parallel mode

npx poku --parallel ./test
```

- Short flag: `-p`.

## API

```ts
await poku('./test', {
  parallel: true,
});
```

<hr />

:::tip
See also the [`concurrency`](/docs/documentation/poku/options/concurrency) option.
:::
