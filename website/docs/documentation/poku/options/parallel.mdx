# `parallel`

Run the tests files in parallel.

## CLI

```bash
# Parallel mode

npx poku --parallel ./test
```

## API

```ts
await poku('./test', {
  parallel: true,
});
```

<hr />

:::tip
See also the [`concurrency`](/docs/documentation/poku/options/concurrency) option.
:::
