# `concurrency`

When using `parallel` option, use `concurrency` to limit the number of tests running concurrently.<br />
The default value is `0` (no limit).

## CLI

```bash
npx poku --parallel --concurrency=4 ./test
```

## API

```ts
await poku('./test', {
  parallel: true,
  concurrency: 4,
});
```
