## **Install**

<Tabs>
  <TabItem default value='Node.js'>

```bash
npm i -D poku
```

  </TabItem>
  <TabItem  value='TypeScript (Node.js)'>

```bash
npm i -D poku tsx
```

  </TabItem>
  <TabItem  value='Bun'>

```bash
bun add -d poku
```

  </TabItem>
  <TabItem  value='Deno'>

```ts
import { poku } from 'npm:poku';
```

You can also use **Poku** with [**esm.sh**](https://esm.sh):

```ts
import { poku } from 'https://esm.sh/poku';
```

    </TabItem>

</Tabs>

## Quick Start

### API (_in-code_)

<Tabs>
  <TabItem default value='Node.js, TypeScript (Node.js) and Bun'>

    ```ts
    import { poku } from 'poku';

    await poku(['targetDir']);
    ```

  </TabItem>
  <TabItem value='Deno'>

```ts
import { poku } from 'npm:poku';

await poku(['targetDir']);
```

  </TabItem>
</Tabs>

### CLI

<Tabs>
  <TabItem default value='Node.js and TypeScript (Node.js)'>

```bash
npx poku targetDir
```

  </TabItem>
  <TabItem  value='Bun'>

```bash
bun poku targetDir
```

  </TabItem>
  <TabItem  value='Deno'>

```bash
deno run npm:poku targetDir
```

    **Poku** requires these permissions by default:
      - `--allow-read`
      - `--allow-env`
      - `--allow-run`

  </TabItem>
</Tabs>
