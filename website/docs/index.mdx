## **Install**

<Tabs>
  <TabItem default value='Node.js'>
  
```bash
npm install --save-dev poku
```

  </TabItem>
  <TabItem  value='TypeScript (Node.js)'>
  
```bash
npm install --save-dev poku tsx
```

  </TabItem>
  <TabItem  value='Bun'>
  
```bash
bun add --dev poku
```

  </TabItem>
  <TabItem  value='Deno'>
  
```ts
import { poku } from 'npm:poku';
```
  - **Poku** requires these permissions by default:
    - `--allow-read`
    - `--allow-env`
    - `--allow-run`

  </TabItem>
</Tabs>

## Quick Start

### API (_in-code_)

<Tabs>
  <TabItem default value='Node.js, TypeScript (Node.js) and Bun'>
  
    ```ts
    import { poku } from 'poku';

    await poku(['targetDirA', 'targetDirB']);
    ```

  </TabItem>
  <TabItem value='Deno'>

```ts
import { poku } from 'npm:poku';

await poku(['targetDirA', 'targetDirB']);
```

  </TabItem>
</Tabs>

### CLI

<Tabs>
  <TabItem default value='Node.js and TypeScript (Node.js)'>
  
```bash
npx poku targetDirA,targetDirB
```

  </TabItem>
  <TabItem  value='Bun'>

```bash
bun poku targetDirA,targetDirB
```

  </TabItem>
  <TabItem  value='Deno'>

```bash
deno run npm:poku targetDirA,targetDirB
```

  </TabItem>
</Tabs>
