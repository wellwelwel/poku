## Quickstart

### **Install** 📦

[![Install Size](https://packagephobia.com/badge?p=poku)](https://packagephobia.com/result?p=poku)

<Tabs>
  <TabItem default value='Node.js'>

```bash
npm i -D poku
```

  </TabItem>
  <TabItem  value='TypeScript (Node.js)'>

```bash
npm i -D poku tsx
```

  </TabItem>
  <TabItem  value='Bun'>

```bash
bun add -d poku
```

  </TabItem>
  <TabItem  value='Deno'>

```ts
import { poku } from 'npm:poku';
```

You can also use **Poku** with [**esm.sh**](https://esm.sh):

```ts
import { poku } from 'https://esm.sh/poku';
```

  </TabItem>
</Tabs>

### **Test it** 🔬

<Tabs>
  <TabItem default value='test/file.test.js'>

```ts
import { assert } from 'poku';

assert(true, 'Poku will describe it 🐷');
```

  </TabItem>
  <TabItem default value='test/file.test.ts (Deno)'>

```ts
import { assert } from 'npm:poku';

assert(true, 'Poku will describe it 🐷');
```

  </TabItem>
</Tabs>

> Note that these examples use [**ESM**](/docs/examples/cjs-esm), but you can use [**CJS**](/docs/examples/cjs-esm) as well.

### **Run it** 🚀

<Tabs>
  <TabItem default value='Node.js and TypeScript (Node.js)'>

```bash
npx poku
```

  </TabItem>
  <TabItem  value='Bun'>

```bash
bun poku
```

  </TabItem>
  <TabItem  value='Deno'>

```bash
deno run npm:poku
```

    **Poku** requires these permissions by default:
      - `--allow-read`
      - `--allow-env`
      - `--allow-run`

  </TabItem>
</Tabs>

### **That's it** 🎉

- [**See the complete `assert`'s documentation**](/docs/documentation/assert).
- [**See the complete `poku`'s documentation**](/docs/category/poku).

## Available Methods

### Essentials

- [**poku**](/docs/category/poku) (_test runner_)
- [**assert**](/docs/documentation/assert) (_test assertion_)
- [**startScript**](/docs/documentation/startScript) (_run `package.json` scripts in a background process_)
- [**startService**](/docs/documentation/startService) (_run files in a background process_)

### Helpers

- `test`
- `beforeEach` and `afterEach`
- `describe` and `log`
- `listFiles`
- `exit`

[**See the complete documentation**](/docs/category/documentation).
