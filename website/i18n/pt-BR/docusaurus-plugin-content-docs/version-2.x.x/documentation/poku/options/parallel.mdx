# `parallel`

Executa os arquivos de teste em paralelo.

<History
  records={[
    {
      version: '2.2.0',
      changes: [
        <>
          Suporte para a flag curta <code>-p</code>.
        </>,
      ],
    },
  ]}
/>

## CLI

```bash
# Modo paralelo

npx poku --parallel ./test
```

- Flag curta: `-p`.

## API

```ts
await poku('./test', {
  parallel: true,
});
```
