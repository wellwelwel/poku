# Aguardando Portas

Aguarda as portas especificadas se tornarem ativas.

## waitForPort

```ts
import { waitForPort } from 'poku';

await waitForPort(3000, {
  delay: 0,
  interval: 100,
  timeout: 60000,
  host: 'localhost',
});
```
