# `concurrency`

Quando utilizada a opção `parallel`, utilize `concurrency` para limitar o número de testes executando simultaneamente.<br />
O valor padrão é `0` (sem limite).

## CLI

```bash
npx poku --parallel --concurrency=4 ./test
```

## API

```ts
await poku('./test', {
  parallel: true,
  concurrency: 4,
});
```
