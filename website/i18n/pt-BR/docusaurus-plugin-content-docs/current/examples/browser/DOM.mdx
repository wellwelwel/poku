# Virtual DOM

Vamos simular um **Virtual DOM** (Navegador) usando [**Happy DOM**](https://github.com/capricorn86/happy-dom) e, em seguida, testar seus resultados com o **Poku**:

:::tip
Para testar componentes **React** com o DOM configurado automaticamente, use o plugin oficial [**@pokujs/react**](/docs/documentation/plugins/official/react).
:::

```ts
import { assert } from 'poku';
// highlight-start
import { Window } from 'happy-dom';

const window = new Window();
const document = window.document;

document.body.innerHTML = '<div id="name">Poku</div>';

const name = document.querySelector('#name');
// highlight-end

assert.strictEqual(name?.textContent, 'Poku', 'Minha div contém um Poku 🐷');
```

Then:

```bash
npx poku
```
