# `sequential`

Run the tests files sequentially.

## CLI

```bash
# Sequential mode

npx poku --sequential ./test
```

## API

```ts
await poku('./test', {
  sequential: true,
});
```

<hr />

:::tip
Learn how to control the number of concurrent test executions with the [`concurrency`](/docs/3.x.x/documentation/poku/options/concurrency) option.
:::

:::info
Sequential execution processes test files one by one, but doesn't guarantee alphabetical.<br />
